# Official Mapping

This file is a quick map from repository folders to official Python documentation areas.

## Main Mapping

- `foundations/` -> tutorial basics, control flow, data structures, exceptions
- `core_python/` -> classes, function definitions, data model, language reference
- `modules/` -> modules tutorial, import system, `__main__`
- `standard_library/` -> standard library modules
- `engineering/` -> practical library usage such as `argparse`, `logging`, `unittest`, `tempfile`
- `advanced/` -> `asyncio`, `threading`, `queue`, `concurrent.futures`, `functools`
- `packaging/` -> packaging concepts and project structure
- `pfpython/` -> reusable local package examples
- `projects/` -> applied practice built on top of official concepts

## Best Use

- Use `OFFICIAL_RESOURCES.md` to open official docs
- Use `official_tracks/` to choose an official-style study path
- Use the repository files as runnable practice for each official topic
