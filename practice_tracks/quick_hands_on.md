# Quick Hands-On Track

Use this track when you want short, direct, low-friction practice.

## Best Folders

- `foundations/`
- `core_python/`
- `modules/`
- `interview/`

## Best Session Length

- 15 to 30 minutes

## Good Daily Pattern

1. Run one `foundations/` file.
2. Run one `core_python/` file.
3. Change one small part.
4. Write one sentence in the daily log.

## Recommended First Files

- `foundations/01_hello_world.py`
- `foundations/05_loops.py`
- `core_python/01_scope_and_mutability.py`
- `modules/01_imports_and_helpers.py`
- `interview/01_two_sum_dict.py`
