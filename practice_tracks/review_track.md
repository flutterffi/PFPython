# Review Track

Use this track when you want to strengthen memory and reduce forgetting.

## Best Folders

- `practice_system/`
- `tools/`
- previously completed files in any learning folder

## Best Session Length

- 10 to 25 minutes

## Good Review Pattern

1. Run `tools/show_next.py`
2. Open one completed file
3. Rewrite its main idea from memory
4. Compare your version to the original
5. Record one note in the daily log
6. Review weekly progress with the dashboard

## Best Supporting Commands

- `python3 tools/show_next.py`
- `python3 tools/progress_dashboard.py`
- `python3 tools/create_daily_log.py`
- `python3 tools/create_weekly_log.py --week week-01`
