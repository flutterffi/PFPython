# Practice System

This folder helps you turn PFPython into a repeatable training habit.

Use these files together:

- `daily_log_template.md`
- `weekly_review_template.md`
- `challenge_backlog.md`
- `checkpoint_template.md`
- `logs/daily/`
- `logs/weekly/`

## Recommended Loop

1. Pick your path from `STUDY_PATHS.md`
2. Use `tools/suggest_today.py` for a quick daily suggestion
3. Use `tools/mark_complete.py --file ...` after finishing a file
4. Use `tools/create_daily_log.py` before or after practice
5. Fill one daily log after practice
6. Use `tools/create_weekly_log.py --week week-01` every 7 days
7. Complete one weekly review every 7 days
8. Pull one idea from `challenge_backlog.md`

## Why This Matters

Code improves faster when you do more than just run files.

Good practice includes:

- prediction
- modification
- repetition
- reflection
- project application
