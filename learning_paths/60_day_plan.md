# 60 Day Plan

This plan is for balanced growth.

Goal:

- become comfortable with Python syntax and modules
- build standard library instincts
- learn basic testing and project structure
- complete several small projects

## Phase 1: Days 1-15

Work through:

- all `foundations/`
- all `core_python/`

Rules:

- run every file
- modify every file
- rewrite at least 5 files from memory

## Phase 2: Days 16-25

Work through:

- all `modules/`
- all `pfpython/`
- all `packaging/`

Goals:

- understand imports
- understand package layout
- understand `python -m`

## Phase 3: Days 26-38

Work through:

- all `standard_library/`
- all `engineering/01` through `engineering/06`

Goals:

- become fluent with `pathlib`, `json`, `csv`, `sqlite3`
- understand CLI and config basics
- start thinking like a maintainer

## Phase 4: Days 39-45

Work through:

- `engineering/07_pytest_fixtures_intro.py`
- `engineering/08_unittest_basics.py`
- `engineering/09_retry_pattern.py`
- all files in `tests/`

Goals:

- learn setup and fixture thinking
- learn the difference between direct-run tests and test frameworks
- understand retry and reliability basics

## Phase 5: Days 46-52

Work through:

- all `advanced/`

Goals:

- understand async basics
- understand queues and worker patterns
- compare sequential and threaded thinking
- learn simple caching

## Phase 6: Days 53-60

Work through:

- all files in `interview/`
- `projects/01_todo_cli.py`
- `projects/03_study_tracker_sqlite.py`
- `projects/04_habit_tracker_app/`
- `projects/05_study_assistant_app/`
- `projects/06_revision_trainer_app/`

Goals:

- solve short problems quickly
- read multi-file project structure confidently
- begin writing your own variants

## Weekly Review Pattern

Every 7 days:

- rerun `python3 tools/run_everything.py`
- pick 2 old files and refactor them
- add 1 new personal exercise
- write down 3 things you understand better now
